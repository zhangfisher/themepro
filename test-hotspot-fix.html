Hotspot 位置修复测试

测试说明：

1. 点击绿色的 hotspot 元素，观察弹出位置是否基于 hotspot 元素

2. 当弹出层显示时，再次点击 host 元素的背景区域，应该不会触发新的弹出事件

3. 点击右下角红色的 target hotspot，观察弹出位置是否准确对齐

4. 弹出内容应该出现在对应 hotspot 元素的附近，而不是基于整个 host 元素

5. 当 hotspot 弹出层激活时，host 元素的点击和 mouseover 事件应该被忽略

预期行为：

  • ✅ 点击 Hotspot 1 时，弹出内容应出现在该按钮的左上方
  • ✅ 点击 Hotspot 2 时，弹出内容应出现在该按钮的正下方
  • ✅ 点击 Target Hotspot 时，弹出内容应出现在右下角按钮的上方
  • ❌ 弹出内容不应该基于整个 host 元素的位置计算
  • 🔥 新增：当 hotspot 弹出层激活时，点击 host 元素背景应该被忽略
  • 🔥 新增：当 hotspot 弹出层激活时，鼠标进入 host 元素应该被忽略
